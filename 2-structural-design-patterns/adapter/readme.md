We use the Adapter pattern to convert the interface of a class to a different form. <br />
Just like how we use adapters in everyday life.

<img src="./assets/Adapter diagram.png" width=350>

Note 1: The `CaramelFilter` is converting the interface of the `Caramel` class to a different form. <br />
Note 2: Here, the `CaramelFilter` is the adapter and the `Caramel` class is the adaptee.
